# GICS v1.3.4 Documentation Deprecation Ledger

Effective date: 2026-03-19
Branch: `codex/1.3.4`

## Policy

- `1.3.4` is a new iteration, not a hotfix continuation of `1.3.3`.
- Documentation produced for `1.3.3` and earlier is deprecated for new implementation work unless it is explicitly carried forward into the active `1.3.4` document set.
- Deprecated documents are not deleted.
- Deprecated documents remain available as read-only historical evidence.

The following documents remain valid as inherited baseline references until they are superseded in `1.3.4`:

- `docs/FORMAT.md`

## Legacy set

The following categories are legacy for new implementation work in `1.3.4`:

- all `1.3.3` planning docs
- all `1.3.2` and earlier roadmaps
- all pre-`1.3.4` release notes
- all previous-cycle architecture documents
- all API references not explicitly promoted into the `1.3.4` active set

## Practical rule

If a new task for `1.3.4` cannot point to an active document above, the documentation is not prepared enough yet and must be updated before implementation starts.
